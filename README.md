# ✦ Air Draw

**v1.2.0 · by BANANCHIKIREAL**

Рисуй в воздухе руками — камера всё видит! Браузерное приложение с трекингом рук через нейронку (MediaPipe HandLandmarker на GPU).

## Жесты

| Жест | Действие |
|------|----------|
| ☝️ Один палец | Рисовать (работают обе руки) |
| ✌️ Два пальца | Ластик |
| 🖐 Взмах ладонью | Стереть всё |
| 👍 Держать | Отменить штрих |
| 👆 Навести на кнопку | Нажать (кольцо-подтверждение) |

## Функции

- 🖊️ Кисти: неон, маркер, спрей, лазер (тает сам)
- 🌈 Палитра + радужный режим, размер кисти слайдером (пальцем в воздухе!)
- 🪞 Зеркальный режим (калейдоскоп)
- 📐 Магия фигур — кривые круги и линии становятся идеальными
- 🖼 Фоны: камера / чёрный / неоновая сетка / космос
- ⏺ Запись видео процесса (webm)
- 📸 Фото с таймером 3-2-1
- ⚡ Турбо-режим для слабых машин
- 🛋 Режим дивана — трекинг руки издалека
- 🎵 Музыкальная кисть — рисование звучит как терменвокс
- 📱 Работает на телефоне: рисование пальцем по экрану (и мышкой на компе)
- ⚙️ Настройки с выбором видимых кнопок
- 💾 Автосохранение рисунка между сессиями

## Запуск локально

```bash
python -m http.server 8347
# открыть http://localhost:8347
```

Нужен интернет при первом запуске (загрузка нейронки с CDN) и разрешение на камеру.

## Технологии

Ванильный HTML/CSS/JS в одном файле. MediaPipe Tasks Vision (HandLandmarker, GPU-делегат), One Euro фильтр для сглаживания, Canvas 2D, MediaRecorder.
